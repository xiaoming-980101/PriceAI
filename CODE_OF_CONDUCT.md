# 行为准则

PriceAI 希望保持一个友好、务实、尊重事实的协作环境。无论是提交 Issue、Pull Request、文档建议还是采集器修复，都请围绕问题本身沟通。

## 我们鼓励

- 清晰描述问题、复现步骤和预期结果。
- 尊重不同使用场景和技术判断。
- 对采集边界、隐私和安全保持谨慎。
- 用具体样例讨论分类规则和价格解析问题。
- 在意见不同的时候，优先给出可验证的证据。

## 不接受

- 人身攻击、骚扰、歧视或威胁。
- 公开粘贴密钥、后台密码、用户隐私数据或漏洞利用细节。
- 提交绕过验证码、登录墙、WAF 或平台风控的实现。
- 故意误导项目维护者或用户的报价、渠道、采集结果。

## 处理方式

维护者可以编辑、隐藏或删除违反本准则的内容，也可以临时或永久限制相关参与者继续参与项目。

如果你发现安全问题，请优先阅读 [SECURITY.md](./SECURITY.md)，不要在公开 Issue 中披露敏感细节。
