# PriceAI 项目架构

PriceAI 是一个面向 AI 订阅与模型 API 的购买前比价工具。核心思路是把分散在官方页面、卡网渠道、模型 API 文档和中转站公开资料中的信息，整理成用户可以比较、核验和回到原站确认的标准对象。

## 四个公开模块

| 模块 | 作用 |
| --- | --- |
| 官方订阅地区价 | 整理 ChatGPT、Claude、Gemini、Grok 等订阅在公开地区价格页中的价格基准 |
| AI 卡网低价订阅 | 聚合卡网、代充、成品号、卡密、CDK、邮箱等第三方渠道报价 |
| 官方 API / 模型 API | 整理官方 API、模型路由、免费额度、Token Plan 和公开套餐限制 |
| 中转 API 比价雷达 | 比较 API 中转站倍率、充值系数、模型覆盖、可用性和公开披露信息 |

## 数据流

```text
公开来源 / 用户提交 / 站点资料
  -> 采集器、导入器或人工审核
  -> 原始报价、官方价格、模型 API 信息或中转站资料
  -> 标准商品、标准模型、标准站点归类
  -> 前台列表、详情页、指南页和原站跳转
```

## 主要边界

- PriceAI 不卖货、不收款、不承诺渠道售后。
- 公开页面展示来源、原始标题、价格、库存、倍率、更新时间和链接，方便用户回原站核验。
- 官方价格、第三方渠道价、官方 API 和中转 API 不混成同一类风险。
- 缺货、过期或未审核信息不能作为外层最低价或客观榜单依据。
