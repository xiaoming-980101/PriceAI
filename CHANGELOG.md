# Changelog

PriceAI / AI 比价雷达的版本更新记录。

## v1.0.0 - 2026-06-12

AI 比价雷达公开初版。

这是 PriceAI 第一个面向公开使用和源码开放协作的正式版本。V1.0 的目标是把分散在卡网、官方订阅地区价和模型 API 渠道中的 AI 获取成本整理成一个可比较、可追踪、可持续扩展的入口。

### 核心能力

- AI 订阅卡网比价：聚合 100+ AI 订阅相关渠道，支持 ChatGPT、Claude、Gemini、Grok、Google、邮箱、API/CDK 等常见品类。
- 标准商品聚合：将分散商品归并为标准商品，展示有货最低价、渠道数、库存状态和最近更新时间。
- 官方订阅地区价：整理官方订阅在不同地区、不同计划下的价格差异，作为第三方渠道报价的参考基准。
- 模型 API 比价雷达：收录部分官方、公开、免费或高性价比 API 渠道，服务开发者和高频 AI 工具用户。
- AI 订阅与使用指南：围绕卡网渠道、官方订阅、地区价、Apple ID、Visa 卡、模型 API 等常见问题沉淀指南文档。
- 渠道提交与反馈：支持用户提交新渠道线索，对具体报价进行反馈，并在后台处理反馈、渠道和采集状态。
- 自动采集与数据更新：支持多类渠道采集器、定时采集、失败记录、来源状态维护和采集性能优化。

### 源码开放协作

- 项目当前采用源码开放策略，允许学习、研究、个人自用、内部非商业部署和非商业贡献。
- 补充 `CONTRIBUTING.md`、`SECURITY.md`、`CODE_OF_CONDUCT.md`、`DATA_LICENSE.md` 等基础公开文档。
- 欢迎社区通过 Issue / Pull Request 贡献新渠道、修正分类、改进采集器、优化前端体验。
- 欢迎以 Markdown / MDX 形式贡献指南文档，例如 AI 订阅购买经验、地区价说明、API 使用方案、风险识别教程等。

### 已知限制

- 部分渠道存在地区限制、风控或验证码，可能无法稳定采集。
- 商品归类仍在持续优化，少量商品可能存在分类不准确。
- 部分报价可能因原站更新、下架或缓存延迟而存在时间差。
- 模型 API 模块仍处于早期阶段，后续会继续扩充数据源和展示维度。

### 重要说明

PriceAI 不售卖任何商品，不收款，也不为第三方渠道提供担保。站内价格、库存和渠道信息仅供购买前参考，实际价格、库存、售后规则和风险以原平台为准。
